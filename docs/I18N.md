# Internationalization (i18n)

Uses `next-intl` with namespaced JSON messages.

## Structure

- Locale files: `src/locales/<locale>/*.json`
- Config: `src/i18n/config.ts`

Supported locales: `en-US`, `pt-BR`.

## Usage

Client components:

```tsx
import { useTranslations } from "next-intl";

const t = useTranslations("auth");
```

Server components:

```tsx
import { getTranslations } from "next-intl/server";

const t = await getTranslations("auth");
```

## Adding a locale

1. Add a new folder under `src/locales/<locale>`.
2. Update `src/i18n/config.ts`.
3. Add translations for each namespace.
